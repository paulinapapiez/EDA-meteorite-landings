# EDA Meteorite Landings
Shiny dashboard of NASA meteorite landings

## Opis
Interaktywna aplikacja Shiny pokazująca lądowania meteorytów na Ziemi na podstawie danych NASA.  
Użytkownik może filtrować dane według roku, masy meteorytu lub typu meteorytu, a wyniki są wyświetlane na interaktywnej mapie i wykresach.

## Technologie
- R  
- Shiny  
- R Markdown  
- HTML / CSS (opcjonalnie)
